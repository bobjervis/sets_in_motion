Callerlab Basic Level Calls

These calls are organized by the Callerlab call names and numbering, using the Callerlab Basic and Mainstream Definitions document dated November 1, 2009.

Sections are included below for each call, even if there is nothing said in that section.  If nothing is said, then the call should work as you would expect.

1. Circle Left / Circle Right

1.a.: 8 Dancers

If you do not specify a fraction to circle, dancers are to move until the caller gives them something else to do.  In the program, it picks about 3 dancer positions worth of movement and stops there.  It also positions the dancers off just a little so that if you start moving the dancers into thars, or have someone move in so that you gets waves, columns or any other common formation it will not be square to the walls not matter how hard you try.  The only way to get the dancers back to truly squared positions is to have the dancers circle or promenade home.

Real dancers will usually align themselves to walls pretty quickly, just because it is so much easier to keep the formations straight.  As a caller, though, you need to be aware that just because your dancers squared up, you can't rely on such phrases as "head positions" or "those facing side walls" to identify dancers until you get them home again.

1.b.: 4 Dancers

You must specify a fraction when having 4 dancers circle, and the circle can only turn in 1/4 multiples.  While Sets in Motion supports moving the whole set out of alignment with the walls, it will not allow you to move some of the dancers out of alignment while leaving other dancers aligned.

1.c: 2 Dancers

The phrasing "single circle (left/right) FRACTION" is recognized as a 2 dancer circle move.  Some callers consider this improper.  If you have a preferred phrasing, you will have to enter your own definition.

4. Swing

You must write this call as "swing your partner" or "swing your corner".  The single word call "swing" is an Adanvced level call.

This call always ends in dancers facing promenade direction (contrary to the Callerlab definition).  You must add something like "promenade 0/4" to get the dancers to face in.

5.a.: Couples

The Callerlab definition lists a set of starting formations, but promenade is often called from slightly different formations than just the ones given.  The definition in Sets in Motion will try the call from any 2x4 formation, for example, which is not included in the Callerlab definitions.  Note that a couples promenade might not work from many 2x4 starting positions, so the program may fail because some later step in the call cannot be done.

9. Left-Hand Start / Right-Hand Star

Eight person stars are supported, but four person stars are not.  Four person stars are not currently supported.  Four person star choreography is generally limited to singing calls and seems highly directional (the dancers do not star specific fractions and adjust to be square to the other dancers as needed by the following calls), all of which makes it very difficult for this program to handle the call.

10. Pass Thru

The current animation for this call has the dancers literally passing thru one another.  Clearly few real dancers will do this for you.

11. Half Sashay

11.a: Half Sashay

You cannot use the shortened "sashay" for this call.

This is defined as a 2 part call, so you can write "1/2 half sashay" or even "1/2 1/2 sashay".

11.b: Rollaway

You must use the single word "rollaway" for this call.

This is defined as a 2 part call, so you can write "1/2 rollaway".

11.c: Ladies in, men sashay

This is defined in the program as "ANYONE in, ANYONE sashay", so you are not restricted to the identities of the dancers doing each part.  The examples in the Callerlab definition makes it clear that this is the intention.

15. Courtesy Turn

This is implemented as a 180° turn except from a right-hand two-face line where it is a 3/4 turn.  Callerlab clearly says that the turn should continue until the couples are facing into the set.  This says that "courtesy turn" from dancers at home would have them turn a full 360° around!  Sets in Motion will just turn half way.

16. Ladies Chain Family

16.a: Two ladies chain

16.b: Four ladies chain

Ladies chain 3/4 is included under this definition.  The current Callerlab document mentions both 2 and four ladies chain here, but the definition really only describes 4 laides chain 3/4.  Sets in Motion currently does not let you name just two ladies, so 2 ladies chain 3/4 cannot be called currently.  Since there is no usable Callerlab definition for this call variation, even when that capablility is added, you will not be able to call this until a proper definition is entered for it.

23. Grand Square

This definition is described in terms of 'steps' (a total of 32 in all), and when it is fractionalized callers say things like "grand square 12 steps".  The Sets in Motion definition is written as an 8 part call and there does not appear any way to provide a precise 32 part definition.  Each part in the program definition implements four steps in the original definition, one of which is a face in.  However, the other three steps are intended to move the dancer 1 full position forward or backward.  When you examine how the dancers must move to make this call work properly, each forward or backward 'step' must move them 1/3 of a position.

Therefore, the definition of "grand square INTEGER steps" in the program converts this to "N/32 grand square".  So, "grand square 12 steps" becomes "3/8 grand square".  In effect, this definition will only allow you to use numbers of steps that are multiples of 4.

25. Circle to a Line

The Callerlab definition discusses styling only for normal boy-girl couples, even though the description of the dance action is more positional.  The program definition is strictly positional.  The final dancer in the line turns out, but Sets in Motion does not animate hand styling.  How same-sex or sashayed couples would complete this part of the call is left open.

48. Sweep a Quarter

The Callerlab definition describes "Sweep a Quarter" in terms of left or right body flow.  The program definition tries to define this in terms of any curving motion whose radius is greater than zero.  The zero-radius exclusion eliminates simple facing turns, such as a "roll".  It also requires that all dancers share this same motion. 

Traditionally, callers have called "Sweep a 1/4" after a wave "Recycle" and dancers have no trouble doing the call.  However, after carefully reviewing both the Mainstream and Challenge definition for "Recycle", it would appear that you cannot call "Sweep a Quarter" after a recycle from a wave.  In the Mainstream definition the original centers final action is to face in and in the Challenge definition, the final action is a roll for all dancers.

The program definition currently disallows "Sweep a Quarter" after a wave "Recycle".

49. Trade By

50. Touch 1/4

The program definition allows no fraction or any fraction that involves an even number of 1/4 turns.  The call "touch" is the same as "step to a wave".
