The Window Frame

When you launch the Sets in Motion shortcut, the program brings up an initial window like the one pictured here.  For now concentrate on the top row of rectangular labels.  Each of these is a tab that is attached to a specific sub-window.  As you use the program more such tabs will appear for the additional sub-windows that are needed.  At any time you can select a tab by clicking on it.  When you select any tab, it will bring its given sub-window forward and give it control of the keyboard.  You will see a blinking cursor in any text field where you can type in that window, if any.

The two halves of the main window are divided by a thin vertical strip.  If you move the mouse cursor over that strip, the mouse cursor changes to a left-right double arrow.  That signals that you are over the strip and if you press the left mouse button, you can drag the separator bar to the left or right.  This allows you to arrange the window as you wish to give you however much space you like in each portion of the window.

The small square button in the far right portion of the tab area is the "More" button.  This button gives you access to a veriety of additional functions.  If you click on it, the menu shown at the left will appear under the mouse cursor.  Note that this button responds to the left mouse button.  This is not a context menu (which is attached to the right mouse button).  Like all the pop-up menus in Sets in Motion, as you move the mouse cursor over the menu choises, one will highlight.  If you click on the highlighted item, you have selected that one.  If you click outside the menu (so that no items are highlighted), then the menu is closed and no action is taken.

Open file...: Pops up a Windows file open dialog.  You may select a single file.   If the file has the extension .CDF it is opened as a definitions file.  If the file has the extension .DNC it is opened as a dance file.  If the file has any other extension, or none at all, it is treated as an SD output file and is loaded as a dance file.  Note that when parsed and loaded as an SD output file, you cannot save any changes back to the original file.  Instead, you will be offered to create a new dance file with the same name as the original file, but with a .DNC extension.
New Dance: This creates a new dance sub-window in the right-hand half of the main window.  The window has no associated file until you save it the first time, when a Windows save file dialog is opened and you are given the opportunity to select a filename and save the file anywhere on your disk.
Open Definitions Tab: The Definitions tab is selected and brought to the front.  If it has been closed it will be re-opened in the left-hand half of the main window.  If that tab is already selected, this function has no further effect.
Open My Definitions Tab: The My Definitions tab is selected.  If it has been closed it will be re-opened in the left-hand half of the main window.  If that tab is already selected, the function has no further effect.
Open My Library Tab: The My Library tab is selected.  If it has been closed it will be re-opened in the right-hand half of the main window.  If that tab is already selected, the function has no further effect.
Open Preferences Tab: The Preferences tab is selected.  If it has been closed it will be re-opened in the right-hand half of the main window.  If that tab is already selected, the function has no further effect.

If you right-click on any tab, you will be shown a context menu specific to that window.  Any tab will include the "Close" option in its context menu.  Selecting that option will remove the tab from those showing at the top of the window.

If you right-click on either the Definitions or My Definitions tabs, you will see a context menu under the mouse cursor that has just two or three choices on it.  The menu has only two options on it when the corresponding file has no unsaved edits.  Definitions files can be either locked or unlocked.  A locked definitions file is read-only and cannot be modified while it is locked.  If the file is currently unlocked (and can be edited), then the menu at left will be shown.  If you selected the "Lock to prevent editing", any further editing of the file will be disallowed until you unlock the file again.

If the file is currently locked you will see the menu at right.  When you select "Unlock for editing", you may then make modifications to the file.  Noe that the default Definitions file is locked initially.  This precaution is in place to make you think before you edit the default Definitions (though you can obviously unlock it yourself and make whatever edits you like).

When the tab you have right-clicked on is a definitions file with unsaved edits you will see the menu on the left.  Select the Save Definitions item to save any edits to the disk.  Note that if you have unsaved edits, you can still Lock the file against further changes and you can still Save the file while it is locked.  Note that while the file is locked you won't be able to Undo or Redo edits either.

When you right-click on a tab for a definition editor, it will aso offer the Lock/Unlock menu items.  If you select these options from a definition editor tab, you will lock or unlock the entire definitions file, not just the specific definition in the current sub-window.
