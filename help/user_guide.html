User's Guide

This describes the details of how to use each part of the Sets in Motion program.

The program is organized into a single main Window.  Almost all of your work will be contained in this window, which is divided more or less into left and right halves with a strip of tabs across the top.  The tabs over each half of the window describe what available sub-windows you can select to view at any given time.

The general interface and specific operations of the tabs themselves are described in the first section and each distinct sub-window is described in the subsequent sections.
