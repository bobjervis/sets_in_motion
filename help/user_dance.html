Dance Editor

The sub-qindow that you can use to view a list of sequences parsed from a file or in My Library is a Dance Editor.  The only difference between a file editor and the My Library editor is that you see the file name for files, but the name of My Library on disk is hidden.  Otherwise, all Dance Editors are alike.

The image at right shows a sample SD output file after it has been opened.  A Dance Editor will always appear in the right-hand pane of the main window.  Notice that when you parse in an SD file, it will be re-formatted as a full Dance file and if you modify it and save it, it will be saved in DNC format (the native format of Sets in Motion).

A Dance Editor shows, if it was created for a specific file, the file name across the top of the pane.  Otherwise, the rest of the pane is filled with six columns of information.

The left-most column is a simple ID number, which is the number of the sequence in the file.  This number will be shown with the sequence when it is being viewed or editied individually.

The next column is a column of checkboxes.  Clicking on the boxes selects or deselects the corresponding sequence.  Note that in the image, three sequences, numbers 7, 10 and 11 have been checked.  Selecting sequences in this way is handy for printing multiple sequences at once.

The Status column can have several possible values:

  • Ready. All calls in the sequence are valid and the dancers finish in home positions. It is ready to be called. 
  • Failed. At least one call in the sequence failed for some reason.  Open the sequence in a Sequence Editor to see more information.  You should probably not call this sequence until the problems have been fixed.
  • Unresolved. All calls in the sequence are valid.  The dancers are not in home positions.  You could call this sequence if you are prepared to site call the dancers home.
  • Unchecked. The definitions files have been modified in some way since this sequence was last checked.  Whenever you modify a definition, all sequences are reset to this state.  When you look at the sequence in a Sequence Editor, it is automatically re-checked and its status is updated.  You can also manually re-check the sequence from the Dance Editor by right-clicking on the sequence and selecting the appropriate menu choice.  See below for details.

The Level column identifies the declared level of each Sequence.

The Date Created column reports the original creation date of a given sequence.

The Comment column is the value of the comment field for each sequence.

At the bottom of the list of sequences is the phrase "Add another sequence...".  If you click there, a new sequence will be created and added to the end of the list.

Clicking the mouse on any sequence will open the sequence in a Sequence Editor in the left-hand pane of the main window.

Even though the changes you make to sequences are done in Sequence Editors on the left-hand side of the main window, they are all part of the dance file represented in the Dance Editor on the right-hand side of the window.  As a result, when you undo an edit, it will be the last edit regardless of which sequence within the file you changed.  Also, you save the dance file and that saves all changes to all sequences at once for that file.

Context Menu Items

Right-clicking over any sequence will open a context menu for that sequence.

The menu choices that are allowed for that sequence are offered.  The following is the list of all menu items that might appear:

  • Edit this Sequence.  This performs the same action that left-clicking on the sequence would.  It opens the sequence in a Sequence Editor.
  • Print this Sequence.  This item will print the sequence.  A printer dialog box is brought up that specifies the printer to use and some basic printing parameters.  Note that the Page Range will show 'All' pages selected, but this has no effect.
  • Print Selected Sequences.  This item will print all the checked sequences.  A printer dialog box is brought up that specifies the printer to use and some basic printing parameters.  Note that the Page Range will show 'All' pages selected, but this has no effect.
  • Recheck this Sequence.  This item is in Unchecked state.  The calls will be checked and the sequence's status will be updated.
  • Recheck all Sequences.  Any sequences in this editor that are Unchecked are rechecked.  All of their status' will be updated.  Currently, the performance of this function is slow enough that this may take a few seconds on a slower machine or for very large dance files (with a hundred or more sequences).
